# Администрирование парковки

Веб-приложение для администратора автостоянки, разработанное в рамках тестового задания. Система позволяет вести учет владельцев, автомобилей, парковочных мест, а также управлять бронированием, освобождением и оплатой мест.

## Стек технологий

| Компонент | Технология | Описание |
|-----------|------------|----------|
| **СУБД** | PostgreSQL | Основное хранилище данных. |
| **Бэкенд** | Java, Spring Boot, Maven | REST API для бизнес-логики и взаимодействия с БД через JdbcTemplate. |
| **Фронтенд** | Vue.js 3 (Composition API), Vite | Пользовательский интерфейс. |
| **Состояние** | Pinia | Хранилище состояния на фронтенде. |
| **UI** | Element Plus | Фреймворк для отрисовки компонентов. |

## Структура проекта

Проект разделен на три основные папки:

- **`backend/`**: Содержит исходный код приложения Spring Boot.
- **`frontend/`**: Содержит исходный код приложения Vue.js.
- **`database/`**: Содержит SQL команды, которые создают и заполняют таблицы.
